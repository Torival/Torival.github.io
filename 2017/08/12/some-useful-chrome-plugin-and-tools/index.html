some useful chrome plugin and tools

Aug 12, 2017 | Life | Hits

我，作为一个标标准准的程序员，最长使用也最喜欢的浏览器莫过于Chrome了，因为其支持各种强大的插件。但是无奈每次重装系统，或者装双系统，或者来到一个新地方的时候，就是一顿摆弄插件。所以，我来总的列一下我常用的 Chrome 插件，若大家喜欢其中的某一个，赶紧麻溜的装起来。

当然，在装这些插件之前，你必须得学会科学上网，不然还怎么访问谷歌商店。不过不会也没有关系，文章最后推荐的软件，就是一个FQ工具，让你嗨起来。

插件篇

Adblock Plus

Adblock Plus是一款强力广告过滤器， 默认会过滤网页上所有烦人的广告，包括 YouTube, Facebook 上的视频广告、晃眼的横幅广告、弹出窗口、隐性弹出窗口以及更多的广告。

Momentum

Momentum是一款高颜值极简风的新标签页扩展程序，每次你按下Ctrl+T打开一个新页面时，就会是下面这样的一个画风，而且每天都会换一张，每天一个新事物。

Stylish

Stylish是一款可以为网站自定义主题的插件，当然，里面会有很多已经做好的主题，供你选择。下图就是我装的Google自定义主题。

Octotree

Octotree是一个关于Github的插件，安装之后，在浏览托管在Github上的项目时，按下Ctrl+Shift+S，便可弹出左侧的树形结构，更方便查看代码。

Checker Plus for Gmail™

Checker Plus for Gmail™，这是一款关于Gmail的插件，无需打开 Gmail 或 Inbox，即可收到桌面邮件通知，方便地查看、收听或删除邮件，并且支持多账户。

JSONView

JSONView是一款格式化Json数据的工具，有时候当我们对接接口时，不得不观察一下Json数据。而直接显示，有事很凌乱的堆在一起，所以，这个可以较好的格式化话。测试点这里。

Save to Pocket

Save to Pocket是一个可以保存页面的工具，比如在你看博客，看文章时，但由于文章太长，一下阅读不完，那么这个时候，你就可以保存一下。之后有时间再拿出来看看。

工具篇

DevDocs

DevDocs是一个在线的API文档，里面涵盖了很多语言，当然也可以下载。下载之后，及时在没有网的环境下，也可以访问查阅相关API。很好用！

百度脑图

百度脑图一个在线画思维导图的工具，使用非常方便，不用安装，登陆 naotu.baidu.com 就可以开始画思维导图。比较容易上手。

Cmd Markdown

Cmd Markdown一个在线写Markdown的工具，当然，优点和百度脑图一样，不用安装，登录 www.zybuluo.com 就可以开始。而且有一个默认文档，即使你不会Markdown语法，也可以照着开始写。

Lantern

蓝灯[Lantern]是一个翻墙工具。至于使用嘛，默认安装，然后就可以打开新世界了。其余的就靠你自己发现吧。
